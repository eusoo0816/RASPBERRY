程式設計
===
停車場感測(使用兩個超音波感測器，以及一個攝影機，一個LED紅色，一個LED綠色，一個LED黃色)  
1.一開始會亮白燈，當超音波偵測<15CM時，蜂鳴器會叫，紅燈會亮，並每隔20秒傳送ERROR訊息給網頁!直到超音波偵測>18CM，蜂鳴器停止，並亮白燈，其他燈熄滅。  
2.當收到網頁傳達A指令會改亮綠燈，其他燈熄滅。  
3.當綠燈亮起時，當超音波偵測<10CM時，會拍一張照片存取至指定資料夾  
4.收到B指令會蜂鳴器會叫，紅燈會亮，其他燈熄滅。  
5.收到C指令時，當2個超音波偵測距離差>3CM時，黃燈會亮起，其他燈熄滅；當2個超音波偵測距離差<3CM時，所有燈熄滅。
